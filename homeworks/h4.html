Answers to questions

  1. Almost all HTML elements can be considered as boxes.It is wrapped around all html elements. Box model is used in CSS to help with the design and layout.
  2. Block level element usually ocuppies a new line and takes up most of the space that ia available.
  3. The Paragraph element is rendered as a block-level element.
  4. An inline element is opposite of a block level element. It doesn't take up a new line and only use space that is neccessary.
  5. One example of an inline element is the form input element.
  6. Two ways we should include CSS in our pages is either through inline style attribute on html elements or the internal style element on the header section.
  7. The selector is used to select the element you want to style. For example, the id and class are selected using "# and .class".
  8. body {background-color:#0000FF;} this goes inside the internal style element on the head section.
